Parameters
jcamp ((string | ArrayBuffer))
options (object? = {})
Name Description
options.keepRecordsRegExp number (default /^$/) By default we don't keep meta information
options.canonicDataLabels number (default true) Canonize the Labels (uppercase without symbol)
options.canonicMetadataLabels number (default false) Canonize the metadata Labels (uppercase without symbol)
options.dynamicTyping number (default false) Convert numbers to Number
options.withoutXY number (default false) Remove the XY data
options.chromatogram number (default false) Special post-processing for GC / HPLC / MS
options.keepSpectra number (default false) Force to keep the spectra in case of 2D
options.noContour number (default false) Don't calculate countour in case of 2D
options.nbContourLevels number (default 7) Number of positive / negative contour levels to calculate
options.noiseMultiplier number (default 5) Define for 2D the level as 5 times the median as default
options.profiling number (default false) Add profiling information
